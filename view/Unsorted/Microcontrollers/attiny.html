Setting up Arduino IDE for use with ATtinys

You will need:

  • Ubuntu 12.04 - setup will be very similar on other OSes
  • Arduino 1.0.1: http://arduino.cc/en/Main/Software
  • TinyCore 100-15: http://code.google.com/p/arduino-tiny/downloads/list
  • AVR ISP programmer - see AVR ISP programmer building blocks page
  1. Download TinyCore and extract the 'tiny' folder to 'arduino-1.0.1/hardware/tiny'
  2. In the Hardware folder you should now have three folders: 'arduino', 'tiny' and 'tools'
  3. Double check you have the folders bootloader and cores, and the file boards.txt inside the 'tiny' folder
  4. Open boards.txt and select the your AVR programmer by un-commenting the correct line, save and close.
  5. Open Arduino, and you should see the ATtiny boards in the Tools>Board menu

To burn the bootloader (set fuses)

This doesn't actually burn a bootloader to enable any sort of serial sketch uploading as in the case of the ATmega328, it just sets the fuses on the ATtiny:

  1. Select the correct board (we use ATtiny84 with 8Mhz internal crystal, BOD disabled for RFM12pi), select your programmer in Tools>Programmer then Tools>Burn Bootloader
  2. You should now be able to upload sketches to the ATtiny wih ISP programmer
