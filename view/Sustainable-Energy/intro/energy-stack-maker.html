The Energy Stack Maker

Note: The energy stack maker has changed quite a bit since this page was written, it is now a cut down module in the new emoncms with a focus on getting the core stack building functionality working - group sharing and comparison will be added again in the future, with better sharing control and comparison tools.

The energy stack maker is an open source web application for creating a visual way of framing the challenge of sustainable energy. The idea and energy stacks are based on David MacKay's book: http://www.withouthotair.com/

Open source

Then energy stack maker an open source web application written in php and javascript, it uses the same base php code as emoncms and so can be integrated with relative ease to create an application that combines real-time energy monitor based data visualisation and the wider context of energy via the energy stacks
