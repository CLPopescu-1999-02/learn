Energy Modeling and Assessment tools

The following are a collection of open source tools that we have developed as part of this project and through collaboration with others in order to better understand the context of energy and how zero carbon energy systems can work.

Hourly zero carbon energy model
This tool, documentation and source code examples are there as an exercise in understanding the dynamics of an zero carbon energy system on an hourly basis, the matching of supply and demand.
Start by loading the tool here so that the dataset for the model loads in time: http://openenergymonitor.org/energymodel
The following examples and sections are available via the left hand menu:

  • Introduction
  • Energy Overview
  • 1. Variable supply
  • 2. Variable supply and flat demand
  • 3. Variable supply, traditional electricity demand and oversupply
  • 4. Mixed supply and flat demand
  • 5. Variable supply and space heating demand
  • 6. Electric Vehicles
  • 7. All
  • Aggregation
  • ZCB Dataset
  • ZCB web model
  • Python models

The examples include javascript and python source code examples.

 

Realtime uk electricity grid visualisation
This tool visualises electricity supply and demand in the uk at present highlighting the role of different electricity generation sources in providing for overall electricity demand.

Live tool and link to source code:

http://openenergymonitor.org/ukgrid
 

 

 

 

 

Zero carbon energy model for the UK
This model was built with Philip James a researcher on the ZeroCarbonBritain project at the Centre for Alternative Technology. It is based on but not exactly the same as their ZeroCarbonBritain spreadsheet model.

http://zerocarbonbritain.org/energy_model/other/3

 

 

 

 

 

 

MyHomeEnergyPlanner
My Home Energy Planner is an open source energy assessment tool that we have developed in collaboration with CarbonCoop for modeling space heating energy requirements of buildings and assessing the effect of carrying out whole home energy retrofits, aiming for 70-80% space heating energy savings.

https://github.com/emoncms/MyHomeEnergyPlanner
