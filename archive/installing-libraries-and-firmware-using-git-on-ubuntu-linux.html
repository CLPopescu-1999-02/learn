Installing Libraries and Firmware using git on Ubuntu Linux

1) Install git

$ sudo apt-get install git-core

Follow this tutorial on creating SSH keys:
https://help.github.com/articles/generating-ssh-keys

Helpful tip: If doing this via SSH, use cat /home/pi/.ssh/id_rsa.pub to display the SSH key in the terminal window, then select and copy to clipboard with Ctrl-Shift-C so you'll be ready to paste it into the Github settings page.

2) Installing Libraries

Before continuing, ina terminal window, locate your Arduino sketchbook directory.

CD to the libraries folder in sketchbook that should have been created automatically by the Arduino IDE:

$ cd libraries

Download the jeelib library from jeelabs.(contains everything for operating the RFM12 wireless board as well jeenode ports) Recent changes introduce a conflict with GLCDLib. See node/3850 for more information.

$ git clone https://github.com/jcw/jeelib.git

Openenergymonitor emonlib library, contains the ac power calculations for electricity monitoring.
$ git clone https://github.com/openenergymonitor/EmonLib.git

One wire DS18B20 temperature sensor library

$ git clone https://github.com/milesburton/Arduino-Temperature-Control-Library.git

Real time clock library

$ git clone https://github.com/jcw/rtclib.git

LCD Display library for EmonGLCD (see note re. JeeLib above.)

$ git clone https://github.com/jcw/glcdlib.git

Ethercard library for the NanodeRF and Open Kontrol Gateway

$ git clone https://github.com/jcw/ethercard.git

This one can't be done with git as it's not on github. Download the OneWire library from pjrc.com and unzip it into the libraries folder in your sketchbook:

http://www.pjrc.com/teensy/td_libs_OneWire.html

2) Installing Firmware

Again, start by locating your Arduino sketchbook directory in a terminal window.

Create a new folder to hold the firmware. (makes navigation easier)

$ mkdir OpenEnergyMonitor
$ cd OpenEnergyMonitor

Download the emonTxFirmware:

$ git clone https://github.com/openenergymonitor/emonTxFirmware.git

The EmonGLCD Firmware

$ git clone https://github.com/openenergymonitor/EmonGLCD.git

The NanodeRF firmware (if your using a NanodeRF basestation)

$ git clone https://github.com/openenergymonitor/NanodeRF.git
